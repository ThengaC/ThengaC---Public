## Welcome to ThengaC 🎉

**തേങ്ങC / ThengaC** is a specialized compiler designed for **mathematical curve analysis and visualization**. It compiles a custom programming language featuring **Malayalam-inspired syntax**.

> A fun compiler with unique Malayalam keywords that makes mathematical programming intuitive and culturally engaging!

## 🎯 What is ThengaC?

ThengaC is **NOT** a general-purpose programming language. It's **purpose-built** for:

- 📈 **Mathematical Function Definition** — Define curves intuitively
- 📊 **Automatic Graph Generation** — Export SVG graphs automatically  
- 🔢 **Calculus Operations** — Built-in integration & differentiation
- 🎨 **Real-time Visualization** — Plot and analyze instantly

### ✨ Key Features

- **LLVM-Backed** — Native code generation for performance
- **Curve Analysis Engine** — Simpson's Rule integration, numerical differentiation
- **SVG Export** — Publication-ready graphs
- **Malayalam Keywords** — Intuitive, culturally-rooted programming

---

## 📚 Language Highlights

ThengaC uses Malayalam-inspired keywords:

| Keyword | Meaning | Usage |
|---------|---------|-------|
| `pani` | Function | Define functions |
| `parayeda` | Print/Tell | Output values |
| `undakk` | Let/Declare | Variables |
| `curve` | Curve | Math functions |
| `graph_undakk` | Graph | Plot & export |
| `curve_parapp` | Integration | Area under curve |
| `cherivu` | Differentiate | Slope at point |
